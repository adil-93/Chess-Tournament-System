# ♟️ Chess Tournament Management System

A Java-based web application that manages chess tournaments efficiently. The system allows administrators to manage players and tournaments, register players, generate random matches, simulate winners, and display tournament rankings.

## 🚀 Features

- 👤 Player Management (CRUD)
- 🏆 Tournament Management (CRUD)
- ➕ Register Players to Tournaments
- ♟️ Random Match Generation
- 🎲 Automatic Winner Selection
- 🥇 Tournament Rankings
- 💾 MySQL Database Integration

## 🛠️ Tech Stack

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat

## 📂 Project Modules

- Player Management
- Tournament Management
- Match Generation
- Result Management
- Ranking System

## ▶️ How to Run

1. Clone the repository.
2. Import the project into Eclipse.
3. Create the MySQL database.
4. Update database credentials in `DBConnection.java`.
5. Deploy the project on Apache Tomcat.
6. Open `http://localhost:8080/ChessTournamentSystem/`.
